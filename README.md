## ✨ Características

- 🎨 Diseño moderno y responsive (móvil, tablet, desktop)
- 🛍️ Catálogo de productos con imágenes de alta calidad
- 🔥 Sección de ofertas con descuentos destacados
- 🛒 Carrito de compras funcional
- 👤 Sistema de login y registro (simulado)
- 💳 Proceso de pago simulado con animaciones
- 📱 Totalmente responsive
- 🎯 Navegación SPA sin recargas
- 🎨 Paleta de colores fresca y atractiva
- 🧭 Footer completo con newsletter y enlaces

---

## 🚀 Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos con variables CSS, Flexbox y Grid
- **JavaScript (Vanilla)** - Lógica del frontend sin frameworks
- **Font Awesome** - Iconos vectoriales
- **Google Fonts** - Tipografías Poppins y Playfair Display
- **Unsplash** - Imágenes de productos de alta calidad
